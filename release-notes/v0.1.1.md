[クリックズームの見た目と操作感を整えた更新です]

## 変更点
- クリックズームの拡大縮小を前より滑らかに調整
- ズーム倍率を `1.2x` から `2.5x` まで選べるよう追加
- 録画開始直後の1回目クリックが通りやすいよう改善

## 使い方
- `ScreenSeal_plus-macOS.zip` をダウンロードして解凍
- `ScreenSeal_plus.app` を `Applications` フォルダに移動
- 初回起動は `右クリック -> 開く`
- `画面収録` の許可を求められたら許可

## 動作環境
- macOS 14.0 以降
- 録画機能は macOS 15.0 以降

## 注意
- ズーム倍率を高くすると、見える範囲が狭くなります
- 環境によっては初回起動時に警告が出ることがあります
- その場合は `右クリック -> 開く` を使ってください
