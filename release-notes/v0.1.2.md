[スクロールキャプチャと英日切り替えを追加し、スクショまわりの使い勝手を大きく改善した更新です]

## 変更点
- `Scroll Capture` を追加。ウィンドウまたは範囲を自動で縦に取り続け、`step_*.png` と `stitched.png` を含む ZIP 1つにまとめて保存できるようにしました
- アプリ内に `Language` を追加。メニューバー、状態表示、主要な操作文言を `English / 日本語` で切り替えられるようにしました
- スクリーンショットに `Screenshot Scale` を追加。`original / 2x / 1x / 0.5x` から保存倍率を選べるようにしました

## 使い方
- `ScreenSeal_plus-macOS.zip` をダウンロードして解凍
- `ScreenSeal_plus.app` を `Applications` フォルダに移動
- 初回起動は `右クリック -> 開く`
- `画面収録` の許可を求められたら許可
- 必要なら `Language` で `English / 日本語` を切り替え
- スクショは `Capture Mode -> Screenshot` を選ぶ
- 通常スクショは `Screenshot Type -> Single Screenshot` を選び、`Take Screenshot` を押す
- スクロールキャプチャは `Screenshot Type -> Scroll Capture` を選び、`Window` か `Region` を指定して `Take Screenshot` で開始、終わる時に `Stop Scroll Capture` を押す
- 保存倍率は `Screenshot Scale` で選ぶ

## 動作環境
- macOS 14.0 以降
- 録画機能は macOS 15.0 以降

## 注意
- `Scroll Capture` は `Full Display` では使えません。`Window` か `Region` を選んでください
- スクロールキャプチャの保存結果は ZIP です。サムネクリック時の開き方は `Screenshot Click Action` に従います
- 既存の画面収録権限が無いと、スクショや録画は動きません
