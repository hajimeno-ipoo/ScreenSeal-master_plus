画面上の見せたくない部分を、モザイクで隠す macOS メニューバーアプリです。

録画やスクリーンショットのときに、パスワードや個人情報を見えにくくできます。
まずは家で試す用の配布版です。

## ダウンロード
- `ScreenSeal_plus-macOS.zip` をダウンロードして解凍してください

## 使い方
- `ScreenSeal_plus.app` を `Applications` フォルダに移動
- 初回起動は、`右クリック -> 開く`
- `画面収録` の許可を求められたら許可
- メニューバーのアイコンからモザイクウィンドウを追加

## できること
- リアルタイムのモザイク表示
- 複数ウィンドウでの隠し表示
- スクリーンショット保存
- 画面録画保存
- レイアウト保存
- マルチディスプレイ対応

## 動作環境
- macOS 14.0 以降
- 録画機能は macOS 15.0 以降

## 注意
- この版は試用向けです
- 環境によっては初回起動時に警告が出ることがあります
- その場合は `右クリック -> 開く` を使ってください
